### Prerequisites

The order to successfully run the code, the following packages and libraries are required. 
* Google BigQuery API
  ```sh
  pip install google-cloud-bigquery
  ```
* Numpy
  ```sh
  pip install numpy
  ```
* Pandas
  ```sh
  pip install pandas
  ```
* NetworkX
  ```sh
  pip install networkx
  ```
* Matplotlib
  ```sh
  pip install matplotlib
  ```
* Seaborn
  ```sh
  pip install seaborn
  ```
